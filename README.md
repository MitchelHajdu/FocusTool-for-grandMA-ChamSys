# FocusTool

A standalone application for lighting programmers to quickly focus fixtures via OSC control. Optimized for handheld devices such as the Steam Deck, MSI Claw, or a laptop, FocusTool communicates directly with lighting consoles like grandMA3 and ChamSys.

## Overview

FocusTool converts joystick and button input into real-time OSC commands for fast and precise fixture focusing. The interface is designed for efficiency and portability, making it ideal for use during setup, rehearsals, or live shows.

#### ChamSys Limitations
Due to ChamSys being a less open platform compared to grandMA3, certain functionalities have limitations:
- The webremote integration is replaced with a dummy interface
- Manual position and group definition is required for fixture control
- Some real-time features may have reduced functionality
